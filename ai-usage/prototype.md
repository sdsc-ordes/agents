# AI Usage Policy

## Definitions

- **AI Code:** has been produced by AI and underwent light or no human review. aka *Vibe Code*.
- **Human Code:** is owned by a human. This implies the owner has edited and fully understood (not only read) it. They are responsible for it, regardless of how it was originally produced.

## Rules

The {project} project is using AI to a large extent. We encourage the following practices:

- **Protect yourself** by sandboxing your agent.
- **Use our [official agent guidelines](https://github.com/sdsc-ordes/agents)** when contributing with the help of AI agents.
- **AI Code MUST be disclosed.** You must state (through a note in the PR description) the extent that the work was AI Code as well as where it was used.

## This is a prototype

The code in this repository is meant for exploratory work.
It has not necessarily been verified by a human.
